OS/2 Base Video Subsystem
=========================

This is open-source OS/2 Base Video Subsystem replacement. It is binary compatible with original OS/2 BVS.
(Note 2 DLL entires not understanded yet). BVS uses BVH for real acting (not yet).

Files
-----

BVS.LIB 16-bit small memory model library. Not depended on OS. Imports DosLoadModule and DosQProcAddr.

BVHSTUB.C - stub to produce BVH
BVSCALLS.DLL - NE library to user with OS/2 and HX DOS.

BVSCALLS.LIB - DLL import library
